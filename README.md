## Project Overview
The *Smart Study Planner* is a productivity application designed to help students manage their academic workload. It moves beyond a simple list by using JavaScript logic to handle user inputs and generate organized study schedules.

## Why I Built This
As an Engineering student, I realized that time management is the biggest challenge. I developed this tool to practice *Form Handling* and *Conditional Logic*—ensuring that the application reacts differently based on the user's specific goals and deadlines.

## Key Features
* *Dynamic Goal Calculation:* Uses JavaScript to process study hours and prioritize tasks.
* *Responsive Input Forms:* Built with custom CSS to ensure a smooth user experience on both mobile and desktop.
* *Logic-Driven Feedback:* Provides real-time updates to the user based on their progress inputs.
* *Clean UI/UX:* Focused on a distraction-free interface using modern CSS Flexbox/Grid.

## Technical Skills Applied
* *HTML5:* Semantic structure for accessibility.
* *CSS3:* Custom styling, transitions, and responsive media queries.
* *JavaScript (ES6+):* DOM manipulation, event listeners, and mathematical logic.

## How to Run
1. Clone the repository.
2. Open index.html in any modern web browser.
